# CrowdConnect

CrowdConnect is a crowdfunding web application that allows users to create, explore, and support campaigns for various causes like education, health, and community development.

## 🚀 Features

* Create and manage fundraising campaigns
* Browse and search campaigns
* Donate to causes
* User authentication (login/signup)
* Dashboard for tracking activity

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: (add yours if used, e.g., MongoDB)

## 📁 Project Structure

* `public/` → frontend files
* `src/` → backend logic
* `scripts/` → utility scripts
* `server.js` → main server

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Chandinisnwl/CrowdConnect.git
cd CrowdConnect
npm install
node server.js
```
